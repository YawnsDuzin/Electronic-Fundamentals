## 1. 필수 장비

이 커리큘럼의 모든 실습을 진행하기 위해 반드시 준비해야 하는 장비입니다.

### 1.1 멀티미터 (디지털 멀티미터, DMM)

전압, 전류, 저항을 측정하는 가장 기본적인 계측기입니다. 전자공학 학습에서 "눈" 역할을 합니다.

#### 필수 기능

| 기능 | 용도 | 중요도 |
|------|------|--------|
| DC 전압 측정 | 회로 내 전압 확인 | 필수 |
| AC 전압 측정 | 교류 전원 확인 | 필수 |
| DC 전류 측정 | 회로에 흐르는 전류 확인 | 필수 |
| 저항 측정 | 저항값 확인, 단선 확인 | 필수 |
| 통전(Continuity) 테스트 | 연결 상태 확인 (부저 소리) | 필수 |
| 다이오드 테스트 | LED/다이오드 극성 및 동작 확인 | 권장 |
| 커패시턴스 측정 | 커패시터 용량 확인 | 선택 |
| 주파수 측정 | PWM 신호 확인 | 선택 |
| 온도 측정 | 열전대 온도 측정 | 선택 |

#### 선택 기준

- **자동 레인지(Auto-Range)**: 측정 범위를 자동으로 설정해주어 초보자에게 편리
- **True RMS**: AC 측정 시 정확한 값 제공 (입문 단계에서는 필수 아님)
- **백라이트 LCD**: 어두운 환경에서도 읽기 편리
- **홀드(Hold) 기능**: 측정값을 화면에 고정
- **CAT 등급**: CAT II 이상 권장 (안전 등급)

#### 추천 제품 (가격대별)

| 등급 | 제품 예시 | 가격대 | 비고 |
|------|-----------|--------|------|
| 입문용 | 중국산 자동레인지 DMM (ANENG, ZOYI 등) | 1~2만원 | 기본 학습에 충분 |
| 중급 | UNI-T UT61E, OWON OW18B | 3~6만원 | 자동레인지, 정확도 높음 |
| 고급 | Fluke 15B+, Fluke 17B+ | 10~15만원 | 내구성, 정확도, 안전성 최고 |

> **입문자 추천**: 1~3만원대 자동레인지 멀티미터로 시작하세요. 통전 테스트(부저) 기능은 반드시 있어야 합니다. 학습이 깊어지면 Fluke 같은 고급 제품으로 업그레이드를 고려하세요.

#### 참고 자료

- Phase 3의 [멀티미터 사용법](../phase3-circuit-design/06-multimeter.md)에서 실제 사용법을 다룹니다.

### 1.2 브레드보드 + 점퍼선

납땜 없이 회로를 구성할 수 있는 시제품(프로토타이핑) 도구입니다.

#### 브레드보드

| 크기 | 포인트 수 | 가격대 | 용도 |
|------|-----------|--------|------|
| 미니 (SYB-170) | 170홀 | 500~1,000원 | 아주 간단한 회로, 휴대용 |
| 하프 (SYB-400) | 400홀 | 1,000~2,000원 | 소규모 회로, LED 실습 |
| **풀사이즈 (MB-102)** | **830홀** | **1,500~3,000원** | **대부분의 실습에 적합 (권장)** |
| 대형 | 1,660홀+ | 5,000~10,000원 | 복잡한 회로, 다중 모듈 연결 |

> **권장**: 풀사이즈(830홀) 브레드보드 1~2개를 준비하세요. 전원 레일이 양쪽에 있는 제품이 편리합니다.

#### 점퍼선 종류

| 종류 | 용도 | 권장 수량 | 가격대 |
|------|------|-----------|--------|
| M-M (수-수) | 브레드보드 내부 연결 | 40~65개 세트 | 1,500~3,000원 |
| M-F (수-암) | 브레드보드 - 모듈 연결 | 40~65개 세트 | 1,500~3,000원 |
| F-F (암-암) | 모듈 간 직접 연결 | 20~40개 세트 | 1,500~3,000원 |
| 단선 점퍼 키트 | 깔끔한 브레드보드 배선 | 1세트 | 3,000~5,000원 |

> **권장**: M-M, M-F를 각각 한 세트씩은 반드시 준비하세요. 세 종류 모두 포함된 120개 세트(약 3,000~5,000원)가 가성비 좋습니다.

#### 브레드보드 사용 팁

- 저가 브레드보드는 접촉 불량이 잦을 수 있으므로 리뷰 확인 후 구매
- 양면테이프가 뒷면에 붙어 있는 제품이 고정에 편리
- 전원 레일(빨강/파랑 줄)의 연결 구조를 반드시 확인 (중간에 끊어진 제품도 있음)

### 1.3 ESP32 개발보드

이 커리큘럼의 핵심 마이크로컨트롤러입니다.

#### ESP32 주요 보드 비교

| 보드 | 칩 | GPIO | Wi-Fi | BLE | USB | 가격대 | 추천 대상 |
|------|------|------|-------|-----|-----|--------|-----------|
| **ESP32 DevKit V1** | ESP32-WROOM-32 | 34 | O | O | Micro-USB (CP2102) | 5,000~8,000원 | **입문자 최우선 추천** |
| ESP32 NodeMCU-32S | ESP32-WROOM-32 | 38 | O | O | Micro-USB (CP2102) | 5,000~9,000원 | 입문자 추천 (핀 배치 다름) |
| ESP32-S2 | ESP32-S2 | 43 | O | X | USB OTG | 5,000~8,000원 | BLE 불필요 시 |
| **ESP32-S3 DevKitC** | ESP32-S3 | 45 | O | 5.0 | USB-C (내장 USB) | 8,000~15,000원 | **성능/기능 최신형 추천** |
| ESP32-C3 | ESP32-C3 | 22 | O | 5.0 | USB-C | 4,000~7,000원 | 소형/저전력 프로젝트 |
| ESP32-C6 | ESP32-C6 | 30 | 6 | 5.0 | USB-C | 6,000~10,000원 | Wi-Fi 6, Zigbee/Thread 지원 |

#### 보드 선택 가이드

```
입문자이고, 온라인 예제를 따라하고 싶다
  └─> ESP32 DevKit V1 (WROOM-32) ← 가장 많은 예제/자료 보유

최신 기능과 USB-C를 원한다
  └─> ESP32-S3 DevKitC ← 성능 좋고 USB 내장

가장 저렴하게 시작하고 싶다
  └─> ESP32-C3 ← 가장 저렴, GPIO 적음

IoT 프로젝트에 최적화하고 싶다
  └─> ESP32-C6 ← Wi-Fi 6, Thread 지원
```

> **입문자 추천**: **ESP32 DevKit V1**을 추천합니다. 인터넷에 가장 많은 예제와 튜토리얼이 있어 문제 해결이 쉽습니다. 2개 이상 구매하면 하나가 고장나도 학습을 계속할 수 있습니다.

#### ESP32 구매 시 주의사항

- **USB 드라이버 칩 확인**: CP2102 또는 CH340 칩 사용 보드는 별도 드라이버 설치 필요. ESP32-S3/C3의 내장 USB는 드라이버 불필요
- **핀 헤더 납땜 여부**: 핀 헤더가 미납땜 상태로 올 수 있음 (납땜 인두 필요)
- **안테나 종류**: PCB 안테나(내장)가 일반적, 외장 안테나(IPEX)는 장거리 통신용
- **정품 vs 호환품**: Espressif 정품 모듈 탑재 보드 권장, 저가 호환품은 불안정할 수 있음
- **USB 케이블**: 데이터 전송 가능한 케이블 필요 (충전 전용 케이블 사용 불가)

### 1.5 USB 케이블

간과하기 쉽지만 매우 중요한 준비물입니다.

| ESP32 보드 | 필요한 케이블 | 주의사항 |

## 2. 선택 장비

학습을 더 깊이 있게 진행하거나 실전 프로젝트를 수행할 때 유용한 장비입니다.

### 2.1 납땜 인두 (Soldering Iron)

핀 헤더 납땜, 영구 회로 제작, PCB 작업에 필요합니다.

#### 추천 사양

| 항목 | 권장 사양 | 이유 |
|------|-----------|------|
| 온도 조절 | 200~450도 가변 | 부품별 적정 온도 사용 가능 |
| 와트 | 40~60W | 빠른 가열, 적절한 열량 |
| 팁 종류 | B형 (원뿔) + D형 (칼날) | B형: 범용, D형: 넓은 패드 |
| 스탠드 | 안전 거치대 포함 | 화상/화재 방지 필수 |

#### 추천 제품 (가격대별)

| 등급 | 제품 예시 | 가격대 | 비고 |
|------|-----------|--------|------|
| 입문용 | 온도 조절 납땜 인두 세트 (60W) | 1~2만원 | 기본 납땜 작업 가능 |
| 중급 | KSGER T12 스테이션 | 3~6만원 | 빠른 가열, 교체 팁 다양 |
| 고급 | Hakko FX-888D, KSGER T245 | 10~20만원 | 전문가용, 내구성 최고 |

#### 납땜 부속품

| 부속품 | 용도 | 가격대 |
|--------|------|--------|
| 납 (유연납 Sn63/Pb37, 0.5~0.8mm) | 납땜 재료 | 3,000~8,000원 |
| 무연납 (Sn96.5/Ag3/Cu0.5) | 환경 친화적 대안 (녹는점 높음) | 5,000~12,000원 |
| 플럭스 (로진 타입) | 납땜 품질 향상 | 2,000~5,000원 |
| 납땜 심지 (디솔더링 위크) | 납땜 제거 | 1,000~3,000원 |
| 납 흡입기 (솔더 석커) | 납땜 제거 | 2,000~5,000원 |
| 팁 클리너 (황동 울) | 인두 팁 청소 | 1,000~3,000원 |
| 내열 실리콘 작업 매트 | 작업대 보호 | 5,000~15,000원 |
| PCB 고정 클램프 (헬핑핸드) | 납땜 시 회로 고정 | 5,000~20,000원 |

> **입문자 팁**: 온도 조절이 되는 인두를 반드시 선택하세요. 온도 고정형은 부품 손상 위험이 높습니다. 납땜 시에는 반드시 환기를 하고, 납연기를 직접 흡입하지 마세요.

### 2.3 로직 분석기

디지털 통신 신호(I2C, SPI, UART 등)를 캡처하고 분석하는 도구입니다.

| 제품 | 채널 | 샘플레이트 | 가격대 | 비고 |
|------|------|-----------|--------|------|
| **Saleae Logic 호환 클론** | 8CH | 24MHz | **5,000~15,000원** | **가성비 최고, 입문 추천** |
| Saleae Logic 8 (정품) | 8CH | 100MHz | 약 500,000원 | 전문가용, 소프트웨어 우수 |
| DSLogic Plus | 16CH | 400MHz | 100,000~150,000원 | 고성능, 중급 |

#### 필수 소프트웨어

- **PulseView** (sigrok): 무료 오픈소스, Saleae 호환 클론과 호환
- **Saleae Logic 2**: 정품 Saleae 전용, 프로토콜 디코딩 우수

> **입문자 추천**: Saleae 호환 8채널 로직 분석기(1만원 내외)면 I2C, SPI, UART 디버깅에 충분합니다. Phase 4 통신 프로토콜 학습 시 큰 도움이 됩니다.

## 부록: 장비 관리 팁

### 멀티미터 관리
- 사용 후 전원 OFF (배터리 소모 방지)
- 측정 레인지를 모를 때는 가장 높은 레인지부터 시작
- 전류 측정 후 반드시 프로브를 전압 측정 단자로 되돌리기 (고장 방지)
- 프로브 선이 손상되면 즉시 교체

### 브레드보드 관리
- 두꺼운 리드선을 억지로 꽂지 않기 (클립 손상)
- 사용하지 않는 회로는 분리하여 보관
- 전원 연결 전 배선을 항상 다시 확인

### ESP32 관리
- 정전기 주의 (다리를 만지지 않기)
- 핀에 허용 전압(3.3V) 초과 전압 인가 금지
- 사용하지 않을 때는 정전기 방지 봉투에 보관
- 펌웨어 업로드 실패 시: BOOT 버튼을 누른 상태에서 EN(리셋) 버튼 누르기
