# Phase 1: 전기/전자 기초 이론

## 개요

이 Phase에서는 전자 회로를 이해하기 위한 **가장 기본적인 전기 이론**을 학습합니다.
ESP32나 라즈베리파이 같은 마이크로컨트롤러를 다루기 전에 반드시 알아야 할 핵심 개념들을 다룹니다.

전기/전자를 처음 접하는 분도 이해할 수 있도록, 일상적인 비유와 실제 ESP32 프로젝트에서 마주치는 상황을 중심으로 설명합니다.

---

## 학습 목표

이 Phase를 완료하면 다음을 할 수 있습니다:

1. **옴의 법칙**을 이용하여 회로에 필요한 저항값을 직접 계산할 수 있다
2. **전력 계산**을 통해 부품의 안전한 사용 범위를 판단할 수 있다
3. **직렬/병렬 회로**의 차이를 이해하고, LED 연결 방식을 선택할 수 있다
4. **DC와 AC**의 차이를 이해하고, 왜 마이크로컨트롤러가 DC를 사용하는지 설명할 수 있다
5. **GND(접지)**의 개념을 이해하고, 장치 간 연결 시 GND를 올바르게 처리할 수 있다
