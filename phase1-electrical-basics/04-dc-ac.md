## 1. DC(직류)란?

### 개념

**DC (Direct Current, 직류)**는 전류가 **한 방향으로만** 흐르는 전기입니다.
전압이 시간이 지나도 **일정하게** 유지됩니다.

```text
    ┌──────────────────────────────────────────────┐
    │                DC (직류) 파형                  │
    │                                              │
    │  전압(V)                                     │
    │    5V ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─   │
    │    │                                         │
    │    │   항상 일정한 전압                        │
    │    │   항상 같은 방향으로 전류가 흐름           │
    │    │                                         │
    │    0V ──────────────────────────── 시간(t)    │
    │                                              │
    └──────────────────────────────────────────────┘
```

### DC 전원의 예시

| DC 전원 | 전압 | 사용처 |
|---------|:---:|------|
| AA 건전지 | 1.5V | 리모컨, 장난감 |
| 리튬 배터리 | 3.7V | 스마트폰, IoT 기기 |
| USB 전원 | 5V | ESP32, 라즈베리파이 |
| 자동차 배터리 | 12V | 자동차 전장 |
| 노트북 어댑터 출력 | 19V~20V | 노트북 |

### DC의 특징

- 전류가 **한 방향**으로만 흐름 (+ → -)
- 전압이 **일정**함 (시간에 따라 변하지 않음)
- **배터리**, **태양전지** 등이 DC를 생성
- **전자 회로**(마이크로컨트롤러, IC)는 DC로 동작

## 2. AC(교류)란?

### 개념

**AC (Alternating Current, 교류)**는 전류의 방향이 **주기적으로 바뀌는** 전기입니다.
전압이 시간에 따라 **사인파(sine wave)** 형태로 변합니다.

```text
    ┌──────────────────────────────────────────────────────────┐
    │                    AC (교류) 파형                          │
    │                                                          │
    │  전압(V)                                                 │
    │  +311V ─          ╭──╮          ╭──╮                     │
    │         │        ╱    ╲        ╱    ╲                    │
    │         │       ╱      ╲      ╱      ╲                   │
    │         │      ╱        ╲    ╱        ╲                   │
    │    0V ──│─────╱──────────╲──╱──────────╲──── 시간(t)     │
    │         │   ╱              ╲╱            ╲               │
    │         │  ╱                              ╲              │
    │         │ ╱                                ╲             │
    │  -311V ─╱                                  ╰──           │
    │         │                                                │
    │         │◄── 1주기(1/60초) ──►│                          │
    │                                                          │
    │  한국: 220V, 60Hz (1초에 60번 방향이 바뀜)                │
    └──────────────────────────────────────────────────────────┘
```

### AC 전원의 예시

| AC 전원 | 전압 | 주파수 | 사용 지역 |
|---------|:---:|:---:|------|
| 한국 콘센트 | 220V | 60Hz | 한국, 일본(동부 일부) |
| 미국 콘센트 | 120V | 60Hz | 미국, 캐나다 |
| 유럽 콘센트 | 230V | 50Hz | 유럽, 호주 |
| 일본 콘센트 | 100V | 50/60Hz | 일본 |

### AC의 특징

- 전류의 **방향이 주기적으로 바뀜** (+ ↔ -)
- 전압이 **사인파** 형태로 변함
- **발전소**에서 AC를 생성하여 송전
- **장거리 전력 전송**에 유리 (변압기로 전압 조절 가능)
- 가정용 콘센트가 AC

### 한국의 220V는 실제로 어떤 뜻인가?

한국 콘센트의 220V는 **실효값(RMS)**입니다.

```text
    실제 피크 전압 = 220V × √2 ≈ 311V

    즉, 콘센트 전압은 -311V ~ +311V 사이를 오가며,
    "평균적인 효과"가 220V와 같다는 뜻입니다.

    ※ 생명에 위험한 전압입니다! 콘센트를 직접 만지면 안 됩니다.
```

## 3. DC와 AC의 비교

```text
    ┌────────────────────────────────────────────────────────┐
    │                DC vs AC 비교표                          │
    ├──────────┬──────────────────┬──────────────────────────┤
    │   항목    │     DC (직류)     │     AC (교류)            │
    ├──────────┼──────────────────┼──────────────────────────┤
    │ 전류 방향 │ 한 방향           │ 주기적으로 바뀜           │
    │ 전압 변화 │ 일정              │ 사인파로 변동            │
    │ 생성 방법 │ 배터리, 태양전지   │ 발전기                  │
    │ 전송 거리 │ 짧은 거리          │ 긴 거리 (변압 가능)      │
    │ 사용 예   │ 전자 회로, 모터    │ 가정 전원, 산업 전력     │
    │ 위험도    │ 저전압은 비교적 안전│ 고전압, 매우 위험        │
    │ 저장     │ 배터리에 저장 가능  │ 직접 저장 불가           │
    │ 변환     │ DC→AC: 인버터     │ AC→DC: 정류기+레귤레이터 │
    └──────────┴──────────────────┴──────────────────────────┘
```

### 왜 세상에는 DC와 AC가 모두 존재할까?

```text
    ┌────────────────────────────────────────────────────────────┐
    │                                                            │
    │  발전소에서 AC를 만드는 이유:                                │
    │  → 변압기로 전압을 쉽게 올리고 내릴 수 있음                  │
    │  → 높은 전압으로 먼 거리를 효율적으로 전송                   │
    │                                                            │
    │  전자 기기가 DC를 쓰는 이유:                                │
    │  → 반도체(트랜지스터, IC)가 DC에서 동작                     │
    │  → 일정한 전압이 필요 (0V = LOW, 3.3V = HIGH)               │
    │  → 배터리가 DC를 제공                                      │
    │                                                            │
    │  결론:                                                     │
    │  발전+송전은 AC ──(어댑터/충전기)──▶ 전자기기는 DC          │
    │                                                            │
    └────────────────────────────────────────────────────────────┘
```

## 4. 왜 ESP32/라즈베리파이는 DC만 다루는가?

### 이유 1: 디지털 논리가 DC 기반이다

ESP32 같은 마이크로컨트롤러는 **디지털 논리**로 동작합니다.

```text
    디지털 신호 (DC 기반):

    전압
    3.3V ──┐     ┌──┐     ┌──────
           │     │  │     │
           │     │  │     │
    0V ────┴─────┴──┴─────┴──────── 시간

    HIGH = 3.3V (1)
    LOW  = 0V   (0)

    → 전압이 일정한 DC여야만 정확한 HIGH/LOW 구분이 가능
    → AC처럼 전압이 계속 변하면 디지털 논리가 작동하지 않음
```

### 이유 2: IC(집적회로)의 동작 원리

```text
    ┌────────────────────────────────────────────────┐
    │  IC 내부의 트랜지스터 (MOSFET)                   │
    │                                                │
    │  게이트 전압에 따라:                             │
    │  - 3.3V (HIGH) → 전류 흐름 (ON)                │
    │  - 0V (LOW) → 전류 차단 (OFF)                  │
    │                                                │
    │  이것이 가능하려면:                              │
    │  → 기준이 되는 일정한 전압 (Vcc = 3.3V)         │
    │  → 기준이 되는 GND (0V)                        │
    │  → 즉, DC 전원이 필수!                         │
    └────────────────────────────────────────────────┘
```

### 이유 3: 센서와 통신도 DC 기반

| 동작 | DC 전압 범위 | 설명 |
|------|:---:|------|
| GPIO 출력 | 0V 또는 3.3V | HIGH/LOW 신호 |
| ADC 입력 | 0V ~ 3.3V | 아날로그 전압 측정 (DC) |
| I2C 통신 | 0V ~ 3.3V | 데이터 전송 신호 |
| UART 통신 | 0V ~ 3.3V | 시리얼 통신 |
| PWM 출력 | 0V ~ 3.3V | DC의 ON/OFF를 빠르게 반복 |

> **결론**: ESP32와 라즈베리파이의 세계에서는 **모든 것이 DC**입니다.
> AC는 전원 어댑터(충전기) 안에서 DC로 변환된 후에 들어옵니다.

## 5. AC 어댑터가 하는 일

### 어댑터의 역할

벽 콘센트(AC 220V)에서 ESP32(DC 5V)까지 전기가 전달되는 과정:

```text
    ┌──────────────────────────────────────────────────────────────┐
    │                AC 어댑터(USB 충전기) 내부 동작                 │
    │                                                              │
    │  콘센트        어댑터 내부                       ESP32        │
    │                                                              │
    │  AC 220V  ──▶  ┌─────────────────────────┐  ──▶  DC 5V     │
    │   60Hz         │                         │       안정적      │
    │                │  1. 변압: 220V → 저전압    │                  │
    │  ~~~           │  2. 정류: AC → 맥류 DC    │       ___        │
    │  사인파         │  3. 평활: 맥류 → DC       │      ─────      │
    │                │  4. 안정화: 정확한 5V 출력  │      일정한     │
    │                │                         │      직선        │
    │                └─────────────────────────┘                  │
    └──────────────────────────────────────────────────────────────┘
```

### 단계별 설명

#### 1단계: 변압 (Transformer)

```text
    AC 220V → AC 저전압 (약 7~9V)

    변압기(트랜스포머)를 사용하여 전압을 낮춥니다.

    (참고: 최근 스위칭 방식 어댑터는 다른 원리를 사용하지만
     결과는 동일합니다)
```

#### 2단계: 정류 (Rectifier)

```text
    AC → 맥류 DC (Pulsating DC)

    다이오드를 사용하여 전류의 방향을 한쪽으로 통일합니다.

    AC 입력:         정류 후:
    ╭─╮   ╭─╮       ╭─╮ ╭─╮ ╭─╮ ╭─╮
    │  │  │  │       │  │ │  │ │  │ │  │
    ───╯╰───╯╰──  →  ───╯ ╰─╯ ╰─╯ ╰───
                     (마이너스 부분이 플러스로 뒤집힘)
```

#### 3단계: 평활 (Smoothing)

```text
    맥류 DC → 거의 평평한 DC

    커패시터(콘덴서)를 사용하여 울퉁불퉁한 전압을 매끄럽게 합니다.

    정류 후:           평활 후:
    ╭─╮ ╭─╮ ╭─╮       ─────────────────
    │  │ │  │ │  │  →    (약간의 리플 존재)
    ───╯ ╰─╯ ╰───       ─ ─ ─ ─ ─ ─ ─
```

#### 4단계: 안정화 (Regulation)

```text
    거의 DC → 정확한 DC 5V

    전압 레귤레이터를 사용하여 정확한 전압을 유지합니다.
    입력 전압이 약간 변해도 출력은 항상 정확히 5.0V를 유지합니다.

    평활 후:            안정화 후:
    ─ ─ ─ ─ ─ ─ ─  →   ──────────────── (정확히 5.00V)
    (약간 울퉁불퉁)
```

### ESP32가 전원을 받는 경로

```text
    ┌──────────────────────────────────────────────────────────┐
    │                                                          │
    │  콘센트(AC 220V)                                         │
    │      │                                                   │
    │      ▼                                                   │
    │  USB 충전기/어댑터 (AC→DC 변환)                           │
    │      │                                                   │
    │      ▼                                                   │
    │  USB 케이블 (DC 5V 전달)                                 │
    │      │                                                   │
    │      ▼                                                   │
    │  ESP32 보드의 USB 커넥터                                  │
    │      │                                                   │
    │      ▼                                                   │
    │  ESP32 보드 내부 레귤레이터 (5V → 3.3V 변환)              │
    │      │                                                   │
    │      ▼                                                   │
    │  ESP32 칩 (DC 3.3V로 동작)                               │
    │                                                          │
    └──────────────────────────────────────────────────────────┘
```

### 라즈베리파이의 경우

```text
    콘센트(AC 220V)
        │
        ▼
    USB-C 충전기 (AC→DC, 5V 3A 출력)
        │
        ▼
    라즈베리파이 보드 (DC 5V로 동작)
        │
        ▼
    보드 내부에서 3.3V, 1.8V 등 다양한 전압 생성
        │
        ▼
    GPIO 핀 (3.3V 레벨)
```

## 6. 전압 표기법 읽기

전자 부품이나 어댑터에 적힌 전압 표기를 읽을 줄 알아야 합니다.

### 일반적인 표기

| 표기 | 의미 | 설명 |
|------|------|------|
| DC 5V | 직류 5볼트 | USB 전원, ESP32 입력 |
| DC 3.3V | 직류 3.3볼트 | ESP32 GPIO 레벨 |
| DC 12V | 직류 12볼트 | LED 스트립, 자동차 |
| AC 220V | 교류 220볼트 | 한국 가정용 콘센트 |
| AC 110V | 교류 110볼트 | 미국 콘센트 |

### 어댑터 라벨 읽기

실제 USB 충전기나 어댑터에 있는 라벨 예시:

```text
    ┌──────────────────────────────────────────┐
    │           USB Power Adapter               │
    │                                          │
    │  INPUT:  AC 100-240V ~ 50/60Hz  0.5A     │
    │  OUTPUT: DC 5V ⎓ 2.4A                    │
    │                                          │
    └──────────────────────────────────────────┘

    해석:
    ┌──────────────────────────────────────────────────────┐
    │                                                      │
    │  INPUT (입력):                                       │
    │  - AC 100-240V: 교류 100~240V 범위에서 동작           │
    │    → 전 세계 어디서나 사용 가능                       │
    │  - ~ (물결표시): 교류를 의미                          │
    │  - 50/60Hz: 50Hz와 60Hz 모두 지원                   │
    │  - 0.5A: 입력에서 최대 0.5A 소비                    │
    │                                                      │
    │  OUTPUT (출력):                                      │
    │  - DC 5V: 직류 5볼트 출력                            │
    │  - ⎓ (직류 기호): 직류를 의미                        │
    │  - 2.4A: 최대 2.4A까지 공급 가능                    │
    │  → 최대 공급 전력: 5V × 2.4A = 12W                  │
    │                                                      │
    └──────────────────────────────────────────────────────┘
```

### 전원 기호

```text
    ┌─────────────────────────────────────────────┐
    │              전원 관련 기호                   │
    ├─────────────────────────────────────────────┤
    │                                             │
    │   ⎓  또는  ── : 직류 (DC)를 의미             │
    │                                             │
    │   ~  또는  ∿  : 교류 (AC)를 의미             │
    │                                             │
    │   ⏚  : 접지 (Earth Ground)를 의미            │
    │                                             │
    │   ⊕  : 양극 (Positive)                      │
    │   ⊖  : 음극 (Negative)                      │
    │                                             │
    │   DC 플러그 극성 표기:                        │
    │                                             │
    │   ─(─ ●─(+)─   : 센터 양극 (Center +)       │
    │     안쪽이 +, 바깥쪽이 -                     │
    │     (가장 일반적인 방식)                      │
    │                                             │
    │   ─(+ ●─(-)─   : 센터 음극 (Center -)       │
    │     안쪽이 -, 바깥쪽이 +                     │
    │     (드물지만 존재함, 주의!)                  │
    │                                             │
    └─────────────────────────────────────────────┘
```

### ESP32/라즈베리파이에서 자주 보는 전압

```text
    ┌─────────────────────────────────────────────┐
    │   ESP32 보드에서 사용하는 전압들              │
    ├─────────────────────────────────────────────┤
    │                                             │
    │   USB 입력:    DC 5V                        │
    │   VIN 핀:     DC 5V (USB에서 직접 연결)      │
    │   3V3 핀:     DC 3.3V (레귤레이터 출력)      │
    │   GPIO 핀:    DC 0V (LOW) / 3.3V (HIGH)     │
    │   ADC 입력:   DC 0V ~ 3.3V                  │
    │                                             │
    │   주의: GPIO 핀에 5V를 인가하면 손상됨!       │
    │   주의: 3.3V 핀에 외부 전원을 연결하지 말 것! │
    │                                             │
    ├─────────────────────────────────────────────┤
    │   라즈베리파이 보드에서 사용하는 전압들       │
    ├─────────────────────────────────────────────┤
    │                                             │
    │   USB-C 입력:  DC 5V 3A (15W)               │
    │   5V 핀:      DC 5V                         │
    │   3.3V 핀:    DC 3.3V                       │
    │   GPIO 핀:    DC 0V (LOW) / 3.3V (HIGH)     │
    │                                             │
    └─────────────────────────────────────────────┘
```

## 7. 핵심 정리

```text
┌──────────────────────────────────────────────────────────────┐
│                         핵심 정리                             │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  ★ DC (직류):                                                │
│    - 전류가 한 방향으로만 흐름                                │
│    - 전압이 일정하게 유지됨                                   │
│    - 배터리, USB, 전자 회로가 DC                              │
│    - ESP32, 라즈베리파이는 DC로 동작                          │
│                                                              │
│  ★ AC (교류):                                                │
│    - 전류 방향이 주기적으로 바뀜                              │
│    - 전압이 사인파 형태로 변동                                │
│    - 콘센트, 발전소, 송전이 AC                               │
│    - 한국: AC 220V, 60Hz                                     │
│                                                              │
│  ★ 왜 마이크로컨트롤러는 DC를 사용하는가:                     │
│    - 디지털 논리(HIGH/LOW)에 일정한 전압 필요                 │
│    - 반도체(IC)가 DC에서 동작하도록 설계됨                    │
│    - 센서, 통신 모두 DC 기반                                 │
│                                                              │
│  ★ AC 어댑터의 역할:                                         │
│    - AC 220V → DC 5V (또는 다른 DC 전압) 변환               │
│    - 과정: 변압 → 정류 → 평활 → 안정화                      │
│                                                              │
│  ★ 전압 표기:                                                │
│    - DC 5V = 직류 5볼트                                     │
│    - AC 220V = 교류 220볼트                                 │
│    - ⎓ = DC, ~ = AC                                        │
│    - ESP32 GPIO: DC 3.3V (절대 5V 인가 금지!)               │
│                                                              │
│  ★ 안전 주의:                                                │
│    - AC 220V는 생명에 위험! 절대 직접 다루지 마세요           │
│    - DC 저전압(3.3V, 5V)은 비교적 안전                       │
│    - 어댑터의 INPUT/OUTPUT 표기를 항상 확인                   │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```
