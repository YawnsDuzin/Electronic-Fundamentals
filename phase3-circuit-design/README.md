# Phase 3: 회로 설계 기초 (Circuit Design Fundamentals)

## 개요

Phase 3에서는 전자 부품들을 실제로 연결하여 동작하는 회로를 만드는 방법을 배웁니다.
Phase 1에서 전기의 기본 원리를, Phase 2에서 개별 부품의 특성을 익혔다면,
이제는 그 부품들을 **올바르게 조합하는 기술**을 배울 차례입니다.

회로 설계는 소프트웨어 개발에서 **아키텍처 설계**와 같습니다.
개별 함수(부품)를 알아도, 이들을 올바르게 조합하지 못하면 시스템이 동작하지 않습니다.

---

## 학습 목표

이 Phase를 완료하면 다음을 할 수 있습니다:

| 번호 | 학습 목표 | 소프트웨어 비유 |
|------|----------|---------------|
| 1 | 브레드보드에 ESP32와 부품을 올바르게 배치하고 연결할 수 있다 | 개발 환경(IDE) 설정 |
| 2 | 회로도(Schematic)를 읽고 실제 배선으로 변환할 수 있다 | 설계 문서를 읽고 코드로 구현 |
| 3 | 풀업/풀다운 저항의 역할을 이해하고 적용할 수 있다 | 변수 초기값 설정 |
| 4 | 전압 분배기를 계산하고 레벨 변환에 활용할 수 있다 | 데이터 타입 변환(casting) |
| 5 | 디커플링 커패시터의 필요성을 이해하고 올바르게 배치할 수 있다 | 에러 핸들링/버퍼링 |
| 6 | 멀티미터로 전압, 저항, 통전을 측정하여 회로를 디버깅할 수 있다 | print 디버깅 / 로깅 |

## 선수 지식

이 Phase를 시작하기 전에 다음 내용을 이해하고 있어야 합니다:

- **Phase 1**: 전압, 전류, 저항의 관계 (옴의 법칙)
- **Phase 2**: 저항, LED, 커패시터, 스위치의 기본 특성

## 필요한 준비물

| 준비물 | 수량 | 비고 |
|--------|------|------|
| ESP32 개발보드 | 1개 | DevKitC 또는 호환 보드 |

| 점퍼선 세트 | 1세트 | M-M, M-F 포함 |

| 세라믹 커패시터 (100nF) | 5개 | 104 표기 |
| 전해 커패시터 (10uF) | 2개 | 16V 이상 |
| 멀티미터 | 1개 | 기본형 디지털 멀티미터 |
| USB 케이블 | 1개 | ESP32 연결용 (Micro-USB 또는 USB-C) |

## 학습 순서 안내

```
01-브레드보드  ──→  02-회로도 읽기  ──→  03-풀업/풀다운
                                              │
                                              ▼
06-멀티미터  ←──  05-디커플링  ←──  04-전압 분배기
```

1번부터 6번까지 순서대로 학습하는 것을 권장합니다.
각 장은 이전 장의 내용을 기반으로 하므로, 순서를 건너뛰지 않는 것이 좋습니다.

---

## 소프트웨어 개발자를 위한 비유 요약

> **브레드보드** = 개발/테스트 환경 (프로토타이핑 도구)
> **회로도** = 아키텍처 다이어그램 / UML
> **풀업/풀다운 저항** = 변수 초기값 (undefined 방지)
> **전압 분배기** = 타입 캐스팅 (데이터 형변환)
> **디커플링 커패시터** = 에러 핸들링 / 입력 버퍼링
> **멀티미터** = 디버거 / console.log
