# Phase 2: 핵심 부품 이해

## 개요

Phase 2에서는 ESP32 프로젝트에서 가장 자주 사용하는 **전자 부품**들을 하나씩 배워봅니다.
각 부품의 원리, 사용법, 주의사항을 이해하면 회로를 직접 설계하고 문제를 해결할 수 있는
기초 체력이 생깁니다.

> **소프트웨어 개발자를 위한 비유:**
> Phase 1이 프로그래밍 언어의 기본 문법(변수, 조건문, 반복문)이었다면,
> Phase 2는 **표준 라이브러리**를 배우는 단계입니다.
> 저항은 `Math.min()` 같은 제한 함수, 커패시터는 버퍼, 트랜지스터는 스위치 함수에 비유할 수 있습니다.

### 권장 학습 순서

1. **저항 + LED** (01, 02): 가장 기본이 되는 부품 조합입니다. 함께 배우세요.
2. **커패시터** (03): 전원 안정화와 노이즈 제거의 기초입니다.
3. **다이오드** (04): LED의 일반화된 형태로, 보호 회로의 핵심입니다.
4. **MOSFET + 릴레이** (05, 06): ESP32로 외부 장치를 제어하는 방법입니다.
5. **레귤레이터** (07): 전원 설계의 기초입니다.
6. **데이터시트 읽기** (08): 모든 부품의 정확한 사양을 확인하는 필수 기술입니다.

## 학습 목표

Phase 2를 완료하면 다음을 할 수 있습니다:

- [x] 저항의 색띠 코드를 읽고 적절한 저항값을 선택할 수 있다
- [x] LED에 필요한 전류 제한 저항을 직접 계산할 수 있다
- [x] 커패시터의 종류를 구분하고 용도에 맞게 배치할 수 있다
- [x] 다이오드의 방향을 올바르게 연결할 수 있다
- [x] MOSFET을 사용하여 ESP32로 고전압 장치를 안전하게 제어할 수 있다
- [x] 릴레이 모듈을 ESP32에 연결하고 제어할 수 있다
- [x] 레귤레이터를 사용하여 안정적인 전원을 공급할 수 있다
- [x] 데이터시트에서 필요한 정보를 찾아 읽을 수 있다
