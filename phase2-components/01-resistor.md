## 1. 저항이란?

저항(Resistor)은 **전자 회로에서 가장 기본이 되는 부품**입니다.
전류의 흐름을 방해(제한)하는 역할을 합니다.

```
소프트웨어 비유:
저항은 네트워크의 "대역폭 제한기(rate limiter)"와 같습니다.
데이터(전류)가 너무 많이 흐르지 않도록 제어합니다.
```

### 저항의 외형

```
        색띠들
    ┌──┬──┬──┬──┐
────┤  │  │  │  ├────     ← 축 방향 리드 (Axial Lead)
    └──┴──┴──┴──┘
   리드            리드
  (다리)          (다리)

    ※ 방향 없음! 어느 쪽이든 연결 가능 (무극성)
```

> **중요:** 저항은 극성이 없습니다. 어느 방향으로 연결해도 동일하게 동작합니다.

## 2. 저항의 역할

### 2-1. 전류 제한 (Current Limiting)

가장 흔한 용도입니다. LED에 과도한 전류가 흐르지 않도록 제한합니다.

```
    ESP32 GPIO (3.3V)
        │
        ├──[220Ω]──┤►├── GND
        │          LED
        │
    저항이 없으면 LED에 과전류 → LED 파손!
```

**옴의 법칙 적용:**
```
V = I × R

전압(V) = 전류(I) × 저항(R)

따라서: I = V / R
저항이 클수록 전류는 줄어듭니다.
```

### 2-2. 풀업 저항 (Pull-up Resistor)

버튼이나 스위치를 사용할 때, 입력 핀의 상태를 확실하게 HIGH로 유지합니다.

```
    3.3V
     │
     ├──[10kΩ]──┬── ESP32 GPIO (입력)
     │          │
     │        [버튼]
     │          │
    GND        GND

    버튼 안 누름 → GPIO = HIGH (3.3V)
    버튼 누름   → GPIO = LOW  (GND)
```

> **참고:** ESP32는 내부 풀업 저항을 소프트웨어로 활성화할 수 있습니다.
> (`INPUT_PULLUP` 모드) 하지만 외부 풀업이 더 안정적입니다.

### 2-3. 풀다운 저항 (Pull-down Resistor)

풀업과 반대로, 입력 핀의 상태를 확실하게 LOW로 유지합니다.

```
    ESP32 GPIO (입력)
         │
         ├──[10kΩ]── GND
         │
       [버튼]
         │
        3.3V

    버튼 안 누름 → GPIO = LOW  (GND)
    버튼 누름   → GPIO = HIGH (3.3V)
```

### 2-4. 전압 분배 (Voltage Divider)

두 개의 저항으로 전압을 나눌 수 있습니다.
센서 출력이 5V인데 ESP32 입력이 3.3V까지만 허용할 때 유용합니다.

```
    Vin (5V)
      │
     [R1]  ← 예: 2kΩ
      │
      ├──── Vout (측정점) → ESP32 GPIO
      │
     [R2]  ← 예: 3.3kΩ
      │
     GND

    Vout = Vin × R2 / (R1 + R2)
    Vout = 5V × 3.3kΩ / (2kΩ + 3.3kΩ)
    Vout = 5V × 3.3 / 5.3
    Vout ≈ 3.11V  ← ESP32에 안전!
```

## 3. 저항의 단위

저항의 단위는 **옴(Ohm, Ω)**입니다. 게오르크 옴(Georg Ohm)의 이름에서 유래했습니다.

| 단위 | 기호 | 값 | 예시 |
|------|:----:|-----|------|
| 옴 | Ω | 1Ω | 0.1Ω, 1Ω, 47Ω, 100Ω, 220Ω, 470Ω |
| 킬로옴 | kΩ | 1,000Ω (10³Ω) | 1kΩ, 4.7kΩ, 10kΩ, 100kΩ |
| 메가옴 | MΩ | 1,000,000Ω (10⁶Ω) | 1MΩ, 10MΩ |

### 단위 변환 예시

```
330Ω   = 0.33kΩ
4.7kΩ  = 4,700Ω
1MΩ    = 1,000kΩ = 1,000,000Ω
```

### 회로도 표기

```
미국식:        유럽/국제식:

  ─/\/\/\─      ─┤    ├─
   zigzag         사각형
```

회로도에서 저항은 위 기호로 표시합니다. 이 문서에서는 `[R]` 또는 `[220Ω]`처럼 표기합니다.

## 4. 색띠 코드 읽는 법

저항의 몸통에 있는 **색띠(Color Band)**를 읽으면 저항값을 알 수 있습니다.

### 색상별 숫자 대응표

| 색상 | 숫자 | 승수 (배수) | 허용 오차 |
|------|:----:|:-----------:|:---------:|
| 검정 (Black) | 0 | ×1 (10⁰) | - |
| 갈색 (Brown) | 1 | ×10 (10¹) | ±1% |
| 빨강 (Red) | 2 | ×100 (10²) | ±2% |
| 주황 (Orange) | 3 | ×1,000 (10³) | - |
| 노랑 (Yellow) | 4 | ×10,000 (10⁴) | - |
| 초록 (Green) | 5 | ×100,000 (10⁵) | ±0.5% |
| 파랑 (Blue) | 6 | ×1,000,000 (10⁶) | ±0.25% |
| 보라 (Violet) | 7 | ×10,000,000 (10⁷) | ±0.1% |
| 회색 (Gray) | 8 | - | ±0.05% |
| 흰색 (White) | 9 | - | - |
| 금색 (Gold) | - | ×0.1 | ±5% |
| 은색 (Silver) | - | ×0.01 | ±10% |

> **암기 팁:** "검갈빨주노초파보회흰" (검은 갈비를 빨간 주황색 노란 초록 파란 보라색 회색 흰색으로...)

### 4색띠 저항 읽기

```
    ┌──────────────────────────────────────┐
    │  [1번] [2번] [3번(승수)] [4번(오차)]   │
    │   ↓     ↓      ↓           ↓         │
    │  갈색  검정   빨강        금색         │
    │   1     0    ×100         ±5%         │
    └──────────────────────────────────────┘

    결과: 10 × 100 = 1,000Ω = 1kΩ (±5%)
```

**4색띠 읽는 순서:**
1. **1번째 띠**: 첫 번째 숫자
2. **2번째 띠**: 두 번째 숫자
3. **3번째 띠**: 승수 (곱하기)
4. **4번째 띠**: 허용 오차

### 5색띠 저항 읽기 (정밀 저항)

```
    ┌──────────────────────────────────────────────┐
    │  [1번] [2번] [3번] [4번(승수)] [5번(오차)]     │
    │   ↓     ↓     ↓      ↓           ↓           │
    │  갈색  검정  검정   갈색        갈색            │
    │   1     0     0    ×10          ±1%            │
    └──────────────────────────────────────────────┘

    결과: 100 × 10 = 1,000Ω = 1kΩ (±1%)
```

**5색띠 읽는 순서:**
1. **1번째 띠**: 첫 번째 숫자
2. **2번째 띠**: 두 번째 숫자
3. **3번째 띠**: 세 번째 숫자
4. **4번째 띠**: 승수 (곱하기)
5. **5번째 띠**: 허용 오차

### 자주 쓰는 저항의 색띠 코드

| 저항값 | 4색띠 코드 | 색상 |
|--------|-----------|------|
| 220Ω | 빨-빨-갈-금 | Red-Red-Brown-Gold |
| 330Ω | 주-주-갈-금 | Orange-Orange-Brown-Gold |
| 470Ω | 노-보-갈-금 | Yellow-Violet-Brown-Gold |
| 1kΩ | 갈-검-빨-금 | Brown-Black-Red-Gold |
| 4.7kΩ | 노-보-빨-금 | Yellow-Violet-Red-Gold |
| 10kΩ | 갈-검-주-금 | Brown-Black-Orange-Gold |
| 100kΩ | 갈-검-노-금 | Brown-Black-Yellow-Gold |

### 색띠 읽기 팁

```
어느 쪽에서 읽어야 할까?

    ┌────────────────────────────┐
    │ 갈 검 빨           금     │
    │ ◀── 이쪽부터 읽기   ──▶   │
    │ (색띠 간격이 좁은 쪽)      │
    │                 (오차 띠는 │
    │                  떨어져 있음)│
    └────────────────────────────┘

    ※ 금색/은색은 항상 마지막(오차)에 옵니다.
    ※ 오차 띠는 다른 띠들보다 약간 떨어져 있습니다.
```

## 5. 자주 쓰는 저항값

ESP32 프로젝트에서 가장 많이 사용하는 저항값과 용도입니다:

| 저항값 | 주요 용도 | 사용 예시 |
|--------|----------|-----------|
| **220Ω** | LED 전류 제한 | 빨간 LED (3.3V 전원, ~10mA) |
| **330Ω** | LED 전류 제한 | 초록/파랑 LED (3.3V 전원) |
| **470Ω** | LED 전류 제한 | LED 밝기를 약간 줄이고 싶을 때 |
| **1kΩ** | 범용, 트랜지스터 베이스 | 트랜지스터/MOSFET 게이트 저항 |
| **4.7kΩ** | I2C 풀업 | I2C SDA/SCL 라인 풀업 |
| **10kΩ** | 풀업/풀다운 | 버튼 입력, 센서 풀업/풀다운 |
| **100kΩ** | 고임피던스 분배 | 전압 분배 회로, 바이어스 |

### E12 시리즈 (표준 저항값)

저항은 아무 값이나 만들어지지 않습니다. **표준화된 값**이 있습니다:

```
E12 시리즈 (±10% 허용오차):
1.0, 1.2, 1.5, 1.8, 2.2, 2.7, 3.3, 3.9, 4.7, 5.6, 6.8, 8.2

이 값들에 10의 거듭제곱을 곱합니다:
× 1    → 1.0Ω,  1.2Ω,  ... 8.2Ω
× 10   → 10Ω,   12Ω,   ... 82Ω
× 100  → 100Ω,  120Ω,  ... 820Ω
× 1k   → 1kΩ,   1.2kΩ, ... 8.2kΩ
× 10k  → 10kΩ,  12kΩ,  ... 82kΩ
× 100k → 100kΩ, 120kΩ, ... 820kΩ
```

## 6. 정격 전력

저항은 전류가 흐르면 **열을 발생**합니다. 정격 전력을 초과하면 저항이 타버립니다!

### 전력 계산

```
P = I² × R     (전류의 제곱 × 저항)
P = V² / R     (전압의 제곱 ÷ 저항)
P = V × I      (전압 × 전류)
```

### 일반적인 정격 전력

| 크기 | 정격 전력 | 특징 |
|------|:---------:|------|
| 소형 (1/8W) | 0.125W | 아주 작은 저항, SMD에서 주로 사용 |
| **일반 (1/4W)** | **0.25W** | **가장 많이 사용! 브레드보드에서 주로 사용** |
| 중형 (1/2W) | 0.5W | 전류가 좀 더 흐를 때 |
| 대형 (1W) | 1W | 큰 전류 회로, 크기가 큼 |
| 초대형 (2W+) | 2W 이상 | 전력 회로, 매우 큼 |

### 정격 전력 확인 예제

```
예제: 220Ω 저항에 15mA가 흐를 때

P = I² × R
P = (0.015)² × 220
P = 0.000225 × 220
P = 0.0495W = 약 50mW

→ 1/4W(250mW) 저항으로 충분! ✓
```

```
예제: 10Ω 저항에 200mA가 흐를 때

P = I² × R
P = (0.2)² × 10
P = 0.04 × 10
P = 0.4W

→ 1/4W(250mW) 저항은 부족! ✗
→ 최소 1/2W(500mW) 저항 필요!
```

> **안전 규칙:** 실제 사용 시에는 정격 전력의 **50~70%** 이하로 사용하는 것이 좋습니다.
> 0.25W 저항이라면 약 0.125~0.175W 이하로 사용하세요.

## 7. 실습: ESP32에서 LED 저항값 계산

### 문제

ESP32의 GPIO 핀(3.3V 출력)에 빨간 LED를 연결하려고 합니다.
적절한 전류 제한 저항값은 얼마일까요?

### 필요한 정보

| 항목 | 값 | 설명 |
|------|-----|------|
| V_source | 3.3V | ESP32 GPIO 출력 전압 |
| V_f (LED) | 약 2.0V | 빨간 LED의 순방향 전압 |
| I_LED | 10~20mA | LED의 적정 구동 전류 |

### 계산 과정

```
Step 1: 저항에 걸리는 전압 계산
    V_R = V_source - V_f
    V_R = 3.3V - 2.0V = 1.3V

Step 2: 저항값 계산 (옴의 법칙: R = V / I)

    ● 20mA로 구동할 때:
      R = 1.3V / 0.020A = 65Ω

    ● 10mA로 구동할 때 (밝기 적당, 수명 길어짐):
      R = 1.3V / 0.010A = 130Ω

Step 3: 표준 저항값 선택
    65Ω  → 68Ω 또는 100Ω 선택
    130Ω → 150Ω 또는 220Ω 선택

    ※ 보통 220Ω을 가장 많이 사용합니다!
```

### 회로 연결도

```
    ESP32                    LED          저항
    ┌────────┐
    │        │
    │  GPIO  ├───────────┤►├───[220Ω]───── GND
    │  (예:  │           LED
    │  GPIO2)│         (빨간색)
    │        │
    │   GND  ├─────────────────────────── GND
    │        │
    └────────┘

    전류 흐름: GPIO → LED → 저항 → GND
    (저항은 LED 앞이든 뒤든 상관없습니다)
```

### 색상별 저항값 참고표 (ESP32 3.3V 기준)

| LED 색상 | 순방향 전압 (Vf) | 권장 저항 (10mA) | 권장 저항 (20mA) |
|----------|:---------------:|:---------------:|:---------------:|
| 빨강 | 1.8~2.2V | 150Ω ~ 220Ω | 68Ω ~ 100Ω |
| 주황 | 2.0~2.2V | 150Ω ~ 220Ω | 56Ω ~ 100Ω |
| 노랑 | 2.0~2.2V | 150Ω ~ 220Ω | 56Ω ~ 100Ω |
| 초록 | 2.0~3.0V | 100Ω ~ 220Ω | 33Ω ~ 68Ω |
| 파랑 | 3.0~3.4V | 33Ω ~ 100Ω | - (전압 부족) |
| 흰색 | 3.0~3.4V | 33Ω ~ 100Ω | - (전압 부족) |

> **주의:** 파랑/흰색 LED는 순방향 전압이 높아서 3.3V에서는 20mA를 흘리기 어렵습니다.
> 5V 전원을 사용하거나, 낮은 전류로 구동하세요.

### Arduino 코드 예제

```cpp
// ESP32에서 LED 깜빡이기
const int LED_PIN = 2;  // GPIO 2번에 LED 연결

void setup() {
    pinMode(LED_PIN, OUTPUT);
}

void loop() {
    digitalWrite(LED_PIN, HIGH);  // LED 켜기 (3.3V 출력)
    delay(1000);                  // 1초 대기
    digitalWrite(LED_PIN, LOW);   // LED 끄기 (0V)
    delay(1000);                  // 1초 대기
}
```

### 전력 확인

```
사용 저항: 220Ω
실제 전류: I = 1.3V / 220Ω ≈ 5.9mA

소비 전력: P = I² × R = (0.0059)² × 220 ≈ 0.0077W = 7.7mW

→ 1/4W (250mW) 저항으로 매우 여유 있음 ✓
```

```
│                                                         │

│  7. 정격 전력: 1/4W가 기본, 초과하면 저항이 탐!             │

```
